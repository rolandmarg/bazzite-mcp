# Tool Routing

Use this map to decide which tasks belong to MCP execution and which belong to skill reasoning.

## MCP-first tasks

- current system state
- package search and installation
- service control
- network changes
- Distrobox, Podman, and Quadlet operations
- VM lifecycle management
- desktop screenshots and automation
- Steam library lookup and settings mutation
- docs cache search and changelog retrieval
- audit log query and rollback

## Skill-first tasks

- choosing the right install tier
- deciding when to avoid `rpm-ostree`
- selecting between Distrobox and VM isolation
- constructing troubleshooting plans
- explaining why a Bazzite-native path is preferable
- turning raw MCP output into a platform-appropriate recommendation
